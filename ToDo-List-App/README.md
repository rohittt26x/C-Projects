# To-Do List Application (C Language)

A simple console-based To-Do List application written in C.  
This project demonstrates core C programming concepts and basic file handling.

## Features
- Add new tasks
- View all tasks with numbering
- Delete a task by number
- Tasks are saved to a file and loaded on program restart
- Menu-driven interface

## Concepts Used
- C programming basics
- Conditional statements
- Loops
- Arrays and strings
- File handling (read/write)
- Modular logic

## How to Run
1. Compile the program:
   gcc todo.c
2. Run the executable:
   ./a.exe

Beginner to Intermediate
